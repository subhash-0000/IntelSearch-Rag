# Access Control Standard

## Least Privilege
All service accounts and users must hold only the permissions required for current duties.

## Quarterly Review
- Engineering managers review repository and cloud IAM access each quarter.
- Security team audits privileged roles monthly.

## Offboarding
- Access revocation must happen before employee separation effective date.
- Tokens and API keys linked to departing employees must be disabled the same day.

## MFA
MFA is required for all production systems and admin dashboards.
